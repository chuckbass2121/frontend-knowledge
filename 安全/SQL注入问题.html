    # SQL 注入

    原理就是用户输入的数据变为了逻辑

    加入后台的查询语句是 select * from user where id = {id}

    当用户输入 id 为 1 or 1 = 1

    查询语句就变为 select * from user where id = 1 or 1 = 1
